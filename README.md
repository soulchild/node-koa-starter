# node-koa-starter

> Boilerplate for new [koa.js](http://koajs.com/) projects.

## Installation

```bash
npm install
```

## Development

Start server with automatic reloading:

```bash
npm run dev
```

Browse to [http://localhost:3000](http://localhost:3000).
